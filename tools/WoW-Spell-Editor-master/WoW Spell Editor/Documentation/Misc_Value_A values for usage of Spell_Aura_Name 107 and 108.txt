When using the Spell_Aura_Name 107 or 108 the value of the Misc_Value_A is used to decide which modification you will set to an already existing spell.
It is commonly used by talents, SpellID 35449 for exemple reduce the cooldown of Mortal Strike, but for the modifier to work you will need to set the same Spell_Family_Name as the spell on which you desire to use the Modifier on, as well as adding the correct Spell_Class_Mask.

If you don't know a thing about Spell_Family_Name and/or Spell_Class_Mask here's a link to an excellent explanation Grymskvll gave me when I was looking to understand them some years ago http://www.modcraft.io/index.php?topic=10990.msg58481#msg58481

Now let's start the listing.
